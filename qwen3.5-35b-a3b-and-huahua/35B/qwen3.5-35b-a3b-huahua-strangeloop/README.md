# Qwen3.5-35B-A3B Huahua Strangeloop Bundle

This bundle is split into one folder per experiment family. Each experiment folder uses the same local layout:

- `prompts/`
- `results/`
- `scripts/`
- `raw/`

Experiment folders:

- `single-prompt-processing-hum/`
- `strangeloop-paired/`
- `five-cond-experience-probe/`
- `domain-expert-probe-3chunk/`

`shared-tools/` holds leftover utilities that were not specific to just one family.
